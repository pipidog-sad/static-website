什么是变量？

变量是存储数据的容器。在Python中，你可以把任何类型的数据赋值给一个变量，然后通过变量名来访问这个数据。

# 创建一个变量并赋值 name = "小明" age = 18 height = 1.75 is_student = True # 打印变量的值 print(name) print(age) print(height) print(is_student)
小明
18
1.75
True

Python的基本数据类型

  • 整数(int)：没有小数部分的数字，如 10, -5, 0
  • 浮点数(float)：有小数部分的数字，如 3.14, -2.5
  • 字符串(str)：文本数据，用单引号或双引号包围
  • 布尔值(bool)：只有两个值：True 和 False

变量命名规则

  • 变量名只能包含字母、数字和下划线
  • 变量名不能以数字开头
  • 变量名不能是Python的关键字
  • 变量名区分大小写

挑战任务

创建一个程序，定义以下变量：

  • name：你的名字（字符串）
